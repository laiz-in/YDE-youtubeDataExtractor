Objectives of this project

1. extract 50 videos from 4 youtube channels, which are
   Krish naik       - https://www.youtube.com/user/krishnaik06
   mysirg           - https://www.youtube.com/user/saurabhexponent1
   tulesko          - https://www.youtube.com/c/Telusko
   Hitesh choudhary - https://www.youtube.com/c/HiteshChoudharydotcom

2. extract following details for latest 50 videos from each channels
   * video links
   * comments with commentor name
   * download videos
   * number of likes
   * number of comments
   * title of the video
   * thumbnail of the video

3. design a mySQL schema for each youtubers with required columns
4. design a mongodb collections for storing comments and video thumbnails (thumbnails needs to be in base64 format)
5. for UI create a simple webpage to input the youtuber name to get the details
